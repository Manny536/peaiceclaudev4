## P/NP Square

The P/NP layer enters as the mathematical square beneath `h < 1`.

Under the standard assumption that `P ≠ NP`:

- checking one exploitable branch can be cheap and local,
- guaranteeing global coherence across all branches becomes combinatorial.

The attacker only needs one gap.

The defender must preserve the field.

This produces the asymmetry:

```txt
Attack verification:   local branch check
Defense verification:  global coherence check

V_adv << G_C
```

Where:

- `V_adv` = adversarial verification cost
- `G_C` = global coherence burden

The claim is not that this repository proves `P ≠ NP`.

The claim is that the P/NP distinction gives a rigorous analogy for why total evaluator sovereignty is structurally unavailable.

`h < 1` is the humility term that prevents governance from pretending otherwise.
